# 古典占星命盤簡易排盤程式 (my-astro-app)

這是一個專業級的**古典占星 (Classical Astrology)** 排盤與分析系統，專為占星分析師、愛好者及開發者設計。本程式不僅提供基於瑞士星曆表 (Swiss Ephemeris) 的精確運算，更深度整合了現代 AI 解析指引，將千年占星智慧與當代人工智慧完美結合。

---

## 🌟 核心排盤模式

本系統採全手動精確輸入設計，確保分析的資料基準 100% 符合使用者的意圖。

### 1. 本命命盤分析 (Nativity)
- **精確輸入**：使用者需自行手動填寫精確的出生時間（西元年、月、日、時:分）與所在地（城市或經緯度）。
- **全方位解盤**：生成包含行星強弱、宮位解析、與推運週期的完整報告。

### 2. 卜卦占星分析 (Horary)
- **問事即時排盤**：使用者手動輸入問題提出的精確時間與地點。
- **針對性分析**：專門針對卜卦邏輯優化，幫助使用者在重大決策中獲得星象參考。

## 🛠️ 專業核心技術與演算法

本程式基於中世紀與希臘古典占星理論，實作了極為精密的運算模組：

### 1. 本質力量評估 (Essential Dignities)
- **完整計分系統**：詳細計算並加總行星在所在位置的「廟 (Domicile)」、「旺 (Exaltation)」、「三分 (Triplicity)」、「界 (Term)」、「面 (Face)」得分。
- **傳統界/面設置**：採用埃及界 (Egyptian Terms) 與迦勒底面 (Chaldean Faces) 規則。

### 2. 後天狀態分析 (Accidental Dignities)
- **環境強度**：判斷行星在宮位中的強度（角宮、續宮、果宮）。
- **特殊狀態**：偵測行星是否處於燃燒 (Combustion)、日心 (Cazimi)、逆行 (Retrograde) 或太陽光輝下等關鍵狀態。

### 3. 古典相位與接納 (Aspects & Reception)
- **托勒密相位**：計算傳統五大相位（合相、六分、四分、三分、對分）。
- **互動機制**：自動判定行星間的「接納 (Reception)」與「互容 (Mutual Reception)」，精準解讀行星間的支援關係與主動意願。

### 4. 希臘點 (Greek Lots / Parts)
- **晝夜區分計算**：自動根據出生時間判斷晝夜，並對應調整幸運點 (Lot of Fortune)、精神點 (Lot of Spirit) 等重要點位的計算公式。

### 5. 恆星合相 (Fixed Stars)
- **關鍵星體偵測**：自動篩選行星是否與重要恆星（如 Regulus、Spica、Algol 等）發生合相，並標註容許度。

### 6. 古典推運系統 (Time Lords)
- **法達星限 (Firdaria)**：完整實作法達大限與小限時間表，掌握生命各階段的主管星（Lords）。
- **小限法 (Annual Profections)**：計算當前年齡對應的小限宮位與年度主星。

## 🤖 AI 深度解析整合

本程式獨創 **「深度思考型 AI 古典占星大師解析指引」**：
- **結構化報告匯出**：自動將所有複雜的占星數據轉化為結構化的 Markdown 格式。
- **大師級 Prompt**：內建針對 Deep Thinking LLMs（如 o1, Gemini 2.0 Thinking, DeepSeek-R1）優化的 System Prompt。
- **批判性分析**：引導 AI 使用「第一性原理」進行客觀且具邏輯性的解構，避免模糊且奉承的解釋。

## 🚀 技術棧與環境
- **後端與邏輯**: Python 3.11+
- **網頁框架**: Streamlit
- **占星計算引擎**: pyswisseph (Swiss Ephemeris), flatlib (專為古典占星優化)
- **地理與時區**: geopy, timezonefinder, pytz

## 📦 安裝與運行
```bash
# 安裝依賴
pip install -r requirements.txt

# 運行程式
streamlit run app.py
```
